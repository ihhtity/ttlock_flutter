## 🔧 热重载 vs 热重启

### 热重载（Hot Reload）- 按 `r` 或保存时自动触发
✅ **适用场景**：
- 修改 UI 样式、颜色、布局
- 修改文本内容
- 添加/删除 Widget
- 修改方法实现

❌ **不适用**：
- 修改 main() 函数
- 添加新文件
- 修改类结构

⚡ **速度**：非常快（1-2秒）
💾 **状态**：保持应用状态

---

### 热重启（Hot Restart）- 按 `R`（大写）
✅ **适用场景**：
- 添加了新的 Dart 文件
- 修改了 main() 函数
- 修改了全局变量
- 热重载无效时

❌ **不适用**：
- 修改原生代码（Java/Kotlin/Swift）

⚡ **速度**：中等（5-10秒）
💾 **状态**：重置应用状态

---

### 完全重新构建 - 停止后重新运行
✅ **适用场景**：
- 修改了 pubspec.yaml
- 修改了原生代码
- 添加了新的依赖包
- 热重启无效时

⚡ **速度**：较慢（30-60秒）
💾 **状态**：完全重置

## ⚠️ 常见问题

### Q1: 保存后没有自动刷新？
**解决方案**：
1. 确认使用的是 VS Code
2. 检查是否安装了 Dart 和 Flutter 扩展
3. 确认配置文件 `.vscode/settings.json` 存在
4. 重启 VS Code

### Q2: 热重载后界面没变化？
**解决方案**：
1. 尝试手动按 `r` 键
2. 如果还不行，按 `R` 键热重启
3. 某些更改需要完全重启应用

### Q3: 如何禁用自动保存？
**解决方案**：
修改 `.vscode/settings.json`：
```json
{
    "files.autoSave": "off"
}
```
