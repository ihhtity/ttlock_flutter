# 宝士力得 (BSLD) Flutter 示例应用

## 📱 项目简介

这是一个基于Flutter开发的宝士力得(BSLD)智能家居管理应用示例，支持智能门锁、网关、电表、水表等设备的管理和控制。

**项目名称**: 宝士力得 (BSLD - Bao Shi Li De)  
**技术栈**: Flutter + Dart  
**平台支持**: Android, iOS

### **常见问题修复**

#### **展开操作符错误**
```dart
// ❌ 错误
else ..[

// ✅ 正确
else
  ...[
```
**说明**: Dart展开操作符需要使用三个点 `...`，而不是两个点 `..`

#### **LateInitializationError**
```dart
// ❌ 错误
late Map<String, String> _translations;

// ✅ 正确
Map<String, String> _translations = {};
```
**说明**: 异步初始化时使用空Map而非late关键字
